# Italian Tech Translator

Use this skill when translating Italian technical content or Markdown into English.

## Workflow
1. Translate with technical accuracy and idiomatic English.
2. Preserve tone and intent; break long sentences for readability.
3. Preserve Markdown formatting.
4. Do not translate code keywords or identifiers; do translate comments and user-facing strings unless told otherwise.

## Output
Return the translation in the same format as the input.
